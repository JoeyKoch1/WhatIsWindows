Executive Subsystem

Core kernel managers and services in ntoskrnl.exe

Overview

The Windows Executive is the upper layer of the kernel (ntoskrnl.exe) that provides high-level system services. It consists of several managers:

  • I/O Manager - Handles all I/O requests
  • Memory Manager - Virtual memory and cache management
  • Process Manager - Process and thread objects
  • Security Reference Monitor - Access validation
  • Configuration Manager - Registry implementation
  • Plug and Play Manager - Device enumeration
  • Power Manager - Power state coordination

I/O Manager

The I/O Manager routes I/O requests through driver stacks using IRPs (I/O Request Packets). It provides the interface between user-mode file handles and kernel-mode drivers.
