Windows Registry Structure

Hierarchical configuration database organization

Overview

The Windows Registry is a hierarchical database storing configuration data:

  • Keys - Container objects (like folders)
  • Values - Named data entries (like files)
  • Hives - Top-level registry sections

Root Keys (Hives)

  • HKEY_LOCAL_MACHINE (HKLM) - System-wide configuration
  • HKEY_CURRENT_USER (HKCU) - Current user settings
  • HKEY_CLASSES_ROOT (HKCR) - File associations and COM
  • HKEY_USERS (HKU) - All user profiles
  • HKEY_CURRENT_CONFIG (HKCC) - Hardware profile
