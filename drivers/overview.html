Windows Drivers Overview

Kernel-mode and user-mode driver architecture

Overview

Windows drivers are loadable kernel modules that extend OS functionality. They fall into categories:

  • File System Drivers - NTFS, FAT32, ReFS
  • Device Drivers - Hardware control (graphics, network, storage)
  • Filter Drivers - Modify I/O requests in driver stacks
  • File System Minifilters - Simplified filtering framework

Driver Stack

I/O requests flow through layered driver stacks:

  • File Object - User-mode handle representation
  • File System Driver - High-level file operations
  • Volume Manager - Disk partitioning
  • Class Driver - Device class abstraction
  • Port Driver - Bus-specific operations
  • Miniport Driver - Hardware-specific code
