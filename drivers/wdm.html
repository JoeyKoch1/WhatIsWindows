Windows Driver Model (WDM)

Driver framework for Windows NT family

Overview

WDM is the standard driver model for Windows, introduced in Windows 98/2000. It provides:

  • IRP (I/O Request Packet) based communication
  • Plug and Play support
  • Power management integration
  • Standardized driver entry points
  • Compatible across Windows versions

Driver Entry Point

Every WDM driver exports DriverEntry, called during driver initialization:

  • Initialize driver object
  • Register dispatch routines
  • Create device objects
  • Set up symbolic links
