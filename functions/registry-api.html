Registry API Functions

Registry manipulation and configuration management

Overview

The Windows Registry API provides functions for reading, writing, and managing registry keys and values. These functions are exported by advapi32.dll.

Key Functions

c
HKEY hKey;
RegOpenKeyEx(HKEY_CURRENT_USER, 
  "Software\\MyApp", 0, KEY_READ, &hKey);

char value[256];
DWORD size = sizeof(value);
RegQueryValueEx(hKey, "Setting", NULL, NULL, 
  (LPBYTE)value, &size);

RegCloseKey(hKey);
