Windows API Functions

Core Windows API functions for system programming

Windows provides thousands of API functions for system programming. These functions are organized by functionality and exported from various system DLLs, primarily kernel32.dll, ntdll.dll, and advapi32.dll.

  • Memory Management

    Memory

    Virtual memory allocation, heap management, memory-mapped files, and page protection functions. Includes VirtualAlloc, HeapCreate, and NtAllocateVirtualMemory.

  • Process & Threads

    Processes

    Process creation, thread management, synchronization primitives, and execution context manipulation. Covers CreateProcess, CreateThread, and synchronization APIs.

  • File I/O

    I/O

    File and device I/O operations including CreateFile, ReadFile, WriteFile, asynchronous I/O with I/O completion ports, and file system control.

  • Registry API

    Registry

    Windows Registry manipulation functions including RegOpenKey, RegSetValue, RegQueryValue, and registry transaction support.
