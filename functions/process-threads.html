Process & Thread Functions

Process creation, thread management, and synchronization

Contents

  • Overview
  • CreateProcess
  • Thread Functions
  • Related Components

Overview

Windows provides comprehensive APIs for process and thread management. Processes are containers for execution with isolated address spaces, while threads are schedulable execution units within processes.

  • Process Creation - CreateProcess, CreateProcessAsUser
  • Thread Creation - CreateThread, _beginthreadex
  • Synchronization - Mutexes, Events, Semaphores, Critical Sections
  • Thread Pools - QueueUserWorkItem, CreateThreadpoolWork

CreateProcess

CreateProcess is the primary function for creating new processes in Windows:

c
STARTUPINFO si = { sizeof(si) };
PROCESS_INFORMATION pi;

CreateProcess(
  NULL,
  "notepad.exe",
  NULL, NULL, FALSE,
  CREATE_NEW_CONSOLE,
  NULL, NULL,
  &si, &pi
);

WaitForSingleObject(pi.hProcess, INFINITE);
CloseHandle(pi.hProcess);
CloseHandle(pi.hThread);

Thread Functions

c
DWORD WINAPI ThreadProc(LPVOID param) {
  printf("Thread running\n");
  return 0;
}

HANDLE hThread = CreateThread(
  NULL, 0, ThreadProc, NULL, 0, NULL
);

WaitForSingleObject(hThread, INFINITE);
CloseHandle(hThread);
