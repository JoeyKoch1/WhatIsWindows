File I/O Functions

File operations, device I/O, and asynchronous I/O

Overview

Windows file I/O is built around handles and overlapped (asynchronous) operations. The I/O Manager routes requests through device stacks to appropriate drivers.

CreateFile

c
HANDLE hFile = CreateFileA(
  "test.txt",
  GENERIC_READ | GENERIC_WRITE,
  0, NULL,
  CREATE_ALWAYS,
  FILE_ATTRIBUTE_NORMAL,
  NULL
);

if (hFile != INVALID_HANDLE_VALUE) {
  const char* data = "Hello World";
  DWORD written;
  WriteFile(hFile, data, strlen(data), &written, NULL);
  CloseHandle(hFile);
}
