msvcrt.dll

Microsoft Visual C Runtime Library - Standard C Library Implementation

Overview

msvcrt.dll is the Microsoft Visual C Runtime library that implements the standard C library functions for Windows. It provides functions for string manipulation, memory operations, mathematical computations, file I/O, and process control.

Unlike other system DLLs, msvcrt.dll is primarily used by C/C++ applications compiled with Microsoft Visual C++. It exports standard functions like printf, malloc, strcpy, and many others defined in the C standard library.

C Runtime Functions

Key function categories exported by msvcrt.dll:

  • I/O Functions - printf, scanf, fopen, fread, fwrite
  • String Functions - strcpy, strcat, strlen, strcmp
  • Memory Functions - malloc, free, memcpy, memset
  • Math Functions - sin, cos, sqrt, pow
  • Time Functions - time, localtime, strftime
  • Process Control - exit, abort, system
