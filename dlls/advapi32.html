advapi32.dll

Advanced Windows API - Security, Registry, and Services

Contents

  • Overview
  • Registry Functions

Overview

advapi32.dll (Advanced API 32-bit) provides critical system functions for registry access, security and authentication, service control, and event logging. It is essential for system-level programming and administrative tools.

The DLL interfaces with several kernel components including the Configuration Manager (registry), Security Reference Monitor, and Service Control Manager.

Registry Functions

advapi32.dll exports the primary Win32 Registry API used by applications to read and write configuration data:

  • RegOpenKeyEx - Open a registry key
  • RegCreateKeyEx - Create or open a registry key
  • RegSetValueEx - Set a registry value
  • RegQueryValueEx - Query a registry value
  • RegDeleteKey - Delete a registry key
  • RegEnumKeyEx - Enumerate subkeys

Security Functions

Security-related functions for access control and authentication:

  • LogonUser - Authenticate a user
  • ImpersonateLoggedOnUser - Impersonate a user context
  • OpenProcessToken - Open a process access token
  • AdjustTokenPrivileges - Modify token privileges
  • GetSecurityInfo - Retrieve security descriptor

RegOpenKeyEx

RegOpenKeyEx opens a specified registry key with desired access rights.

Function Signature

LSTATUS RegOpenKeyExA(
  HKEY   hKey,
  LPCSTR lpSubKey,
  DWORD  ulOptions,
  REGSAM samDesired,
  PHKEY  phkResult
);

Parameters

ParameterTypeDescription
hKeyHKEYHandle to open parent key (HKEY_LOCAL_MACHINE, etc.)
lpSubKeyLPCSTRName of subkey to open
ulOptionsDWORDReserved, must be 0
samDesiredREGSAMAccess mask (KEY_READ, KEY_WRITE, KEY_ALL_ACCESS)
phkResultPHKEYPointer to receive key handle

Example: Reading a Registry Value

c
#include 
#include 

int main() {
  HKEY hKey;
  LONG result = RegOpenKeyExA(
    HKEY_LOCAL_MACHINE,
    "SOFTWARE\\Microsoft\\Windows NT\\CurrentVersion",
    0,
    KEY_READ,
    &hKey
  );
  
  if (result == ERROR_SUCCESS) {
    char productName[256];
    DWORD size = sizeof(productName);
    
    RegQueryValueExA(hKey, "ProductName", NULL, NULL, 
                     (LPBYTE)productName, &size);
    
    printf("Windows: %s\n", productName);
    RegCloseKey(hKey);
  }
  
  return 0;
}
